Why Seashark?

⚡ Fast

Runs on a high-performance interpreter and planned compiler backend.

📝 Simple

Beginner-friendly syntax, easy to read and write.

🌐 Versatile

Write apps, games, and websites — all in one language.

Quick Start

git clone https://github.com/YOUR-USERNAME/Seashark.git
cd Seashark
mkdir build && cd build
cmake ..
cmake --build .
./seashark ../examples/hello.ss
