Welcome

Seashark is a new programming language designed to be simple, fast, and powerful. You can build apps, games, and even websites with it.

Get Started

git clone https://github.com/YOUR-USERNAME/Seashark.git
cd Seashark
mkdir build && cd build
cmake ..
cmake --build .
./seashark ../examples/hello.ss
